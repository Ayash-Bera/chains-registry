# SHRAPNEL

## Overview

Shrapnel is an award-winning, free-to-play, team-based first-person shooter set in a post-apocalyptic world. Players collect and secure "Sigma" from meteorites while battling other squads in the dangerous "Sacrifice Zone." Featuring true digital ownership of in-game assets and a dynamic gameplay experience, Shrapnel integrates blockchain technology for item ownership and a thriving in-game economy with the SHRAP token.

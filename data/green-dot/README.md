# Green Dot

## Overview

Cayg.app (Close As You Go) is a secure, cloud-based platform powered by Avalanche's Green Dot L1, designed to streamline disaster recovery documentation, procurement workflows, and policy modeling for government agencies. As part of Deloitte's blockchain solutions, it combines EVM compatibility with enterprise-grade auditing to modernize public-sector operations.

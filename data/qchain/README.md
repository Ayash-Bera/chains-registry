# QCHAIN

## Overview

Quboid is a decentralized loyalty program platform designed to enhance customer engagement by offering tokenized loyalty points, membership passes, and rewards. It facilitates brand-to-brand collaborations, allowing seamless partnerships via smart contracts. With Q-Chain, it ensures fast settlement of token liabilities and offers a scalable and modular solution for businesses to expand their loyalty programs.

# Blitz

## Overview

Blitz.gg is a gaming companion app offering real-time insights, analytics, and personalized advice for games like League of Legends, Valorant, and Teamfight Tactics. Its L1 powers features like subscriptions, in-game currency, Blitz Battle Rewards, and the Marketplace.

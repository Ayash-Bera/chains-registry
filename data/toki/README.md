# Toki

## Overview

Toki is a platform designed to help businesses build and manage customer loyalty programs. It offers customizable solutions including paid memberships, referral programs, rewards systems, and engagement features like gamification. Toki integrates seamlessly with e-commerce platforms, providing businesses with tools to track performance, offer cashback rewards, and create personalized customer experiences.

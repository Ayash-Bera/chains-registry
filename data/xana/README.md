# XANA

## Overview

XANA is a Web3 platform focused on creating metaverse experiences with blockchain integration. It allows users to interact in a virtual world with avatars, digital assets, and NFTs, enabling decentralized and immersive gaming and social activities. XANA aims to build a bridge between reality and the metaverse, offering businesses and users opportunities to engage in virtual economy and community-building.
